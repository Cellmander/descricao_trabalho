Descrição do projeto

Este projeto é um Editor JavaScript Online que permite escrever e executar código JavaScript diretamente no navegador. É uma ferramenta simples com interface amigável, desenvolvida como parte da disciplina de Desenvolvimento Web.

Objetivos

  • Criar interface para edição de código JavaScript
  • Implementar funcionalidades básicas de execução
  • Aplicar conhecimentos de desenvolvimento web
